# Projeto Poções e Soluções

Sistema de gestão de alquimia desenvolvido para Anna Merigold.

## Como executar

1. **Backend**:
   - Entre na pasta `backend`: `cd backend`
   - Instale: `npm install`
   - Execute: `node server.js`
2. **Frontend**:
   - Entre na pasta `frontend`: `cd frontend`
   - Instale: `npm install`
   - Execute: `npm run dev`
